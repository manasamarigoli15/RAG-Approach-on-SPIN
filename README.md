# RAG-Approach-on-SPIN

## Project Overview
#### Automate the extraction of SPM–target protein interactions from PubMed articles using RAG and LLM-based relation extraction.

## Technical Components

#### Data Retrieval: Query PubChem for SPM synonyms and PubMed for full-text articles.
#### Text Processing: Filter, chunk, and embed text containing SPMs
#### Information Extraction: Use RAG and LLMs (Mistral) to extract interaction triplets.
#### Evaluation & Visualization: Assess retrieval (RAGAs) and visualize the interaction graph (Neo4j).
